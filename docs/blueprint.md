# **App Name**: Collabora

## Core Features:

- User Authentication: Secure email/password and optional OAuth (Google/GitHub) authentication with JWT-based sessions.
- Professional Profiles: Public portfolio profiles with bio, headline, skills, categories (tech, design, video, image), availability, and contact options.
- Company Pages: LinkedIn-style company profiles with description, industry, tech/tools stack, posted projects, and open roles.
- Project Listings: Post projects/jobs with title, category, description, required skills, budget range, timeline, and project type (freelance, contract).
- Application System: Professionals apply to projects with a short message and automatically attached portfolio.
- Internal Messaging: Text-only internal chat between companies and professionals for direct communication.
- Smart Category Assignment: AI-powered tool that automatically suggests relevant categories based on project description.

## Style Guidelines:

- Primary color: Deep ocean blue (#2D3E50) for a professional and trustworthy feel.
- Background color: Light cool gray (#E5E7EB) to provide a clean and spacious layout.
- Accent color: Electric cyan (#7DF9FF) for highlights and CTAs, giving a modern and tech-forward look.
- Headline font: 'Space Grotesk' (sans-serif) for a techy and modern feel. Body font: 'Inter' (sans-serif) for readability and clean aesthetics.
- Use clean, outlined icons with a touch of cyan to represent different categories and actions.
- SaaS-style layout with strong whitespace, clear sections, and a focus on usability and clarity.
- Subtle transitions and animations for user interactions, enhancing the overall user experience without being distracting.
